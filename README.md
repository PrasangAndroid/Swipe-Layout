# Android Swipe Layout
## Usage

### Step 1
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
#### Gradle

```groovy


	dependencies {
	        implementation 'com.github.PrasangAndroid:Swipe-Layout:-SNAPSHOT'
	}

```

#### Maven
<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>

```xml
<dependency>
	    <groupId>com.github.PrasangAndroid</groupId>
	    <artifactId>Swipe-Layout</artifactId>
	    <version>-SNAPSHOT</version>
	</dependency>
